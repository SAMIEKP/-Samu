# -Samu-main 🚀

## Overview
This repository is a personal portfolio/collection of learning and exercise projects across **AI/Data Engineering**, **Cybersecurity**, **Web Development**, and mathematical problem-solving. It aggregates hands-on coding assignments and tools for skill-building.

## 📁 Directory Structure

| Directory | Description | Tech Stack | How to Run |
|-----------|-------------|------------|------------|
| [AI_DATA_ENGINEERING/](AI_DATA_ENGINEERING/) | Python scripts for data analysis, binary classification (Iris), housing data, matrix ops, optimization problems (Fuji descent, paper folding), and more. | Python | `cd AI_DATA_ENGINEERING/ && python <script.py>` (e.g., `python Iris_Data_Analysis.py`) |
| [CyberSecurity_Pro/](CyberSecurity_Pro/) | Ethical hacking tools including Batch Multi-Tool (with Putty, Wireshark, VBox links) and Visual Basic scripts. | Batch (.bat), VBS | `cd CyberSecurity_Pro/Ethical_Hacking/Batch-Multi-Tool-main/multitool/ && main.bat` |
| [DIC coffee/](DIC coffee/) | Static cafe website (blueberry pancakes theme) with menu, contact, responsive design. | HTML, CSS | Open `index.html` in browser: `open "DIC coffee/index.html"` |
| [New_Folder/](New_Folder/) | AI assistance Python script and Python notes. | Python | `cd New_Folder/ && python AI assitance.py` |
| [New_Pro/](New_Pro/) | Additional assignment files. | Markdown | View `github_task.md` |
| [Paper_Folding/](Paper_Folding/) | Paper folding math problem solution. | Unknown | Check `Folding the paper` |

## 🛠️ Prerequisites
- Python 3.x for AI/Data scripts.
- Browser for web project.
- Windows/Command prompt for Batch tools (or compatible env).

## 📝 Additional Notes
- Root `github_task.md`: Assignment tracker.
- Run Python scripts in respective dirs (no external deps noted).
- Explore subdirs for individual project details.
